# Data licensing & attribution

LiftLab's **code** is MIT-licensed (see [LICENSE](LICENSE)). This file governs the
**data substrate** only, and clarifies the boundary between the two.

## Olist Brazilian E-Commerce dataset (optional real population)

- **Dataset:** Brazilian E-Commerce Public Dataset by Olist
- **Creator / attribution:** Olist
- **Source:** https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
- **License:** Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International
  (**CC BY-NC-SA 4.0**), license deed: https://creativecommons.org/licenses/by-nc-sa/4.0/
- **Modifications:** When used, the data is **transformed** by LiftLab's DuckDB + dbt
  pipeline (typed staging views and the `customer_metrics` aggregate mart). A
  synthetic randomized-experiment layer is overlaid on top of it.

### What this means for LiftLab

- **Non-commercial only.** Because the Olist data is NC-licensed, any use of LiftLab
  with the real Olist substrate must remain **non-commercial**.
- **ShareAlike.** Derivatives of the Olist *data* inherit CC BY-NC-SA 4.0. This applies
  to the data and data-derived artifacts, **not** to LiftLab's MIT-licensed code.
- **No redistribution here.** This repository does **not** redistribute the Olist data:
  the raw CSVs are git-ignored and fetched locally via the Kaggle API on demand. When
  Kaggle is unavailable, LiftLab generates a **disclosed synthetic stand-in** instead
  (recorded in `data/raw/MANIFEST.json`), which is original, non-Olist data.

### Scope boundary (MIT code vs. NC/SA data)

| Artifact | License |
| --- | --- |
| LiftLab source code, dbt models, configs, docs | MIT (this repo) |
| Real Olist CSVs (if you fetch them) and data-derived outputs | CC BY-NC-SA 4.0 (Olist) |
| LiftLab's synthetic population (the offline fallback) | MIT (original, generated by this repo) |

If you only ever run the synthetic fallback, no CC BY-NC-SA terms attach to anything.
